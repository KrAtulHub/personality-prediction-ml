# Personality Prediction using Machine Learning

## 📌 Project Overview
This project predicts a person's **personality type** using behavioral and lifestyle features such as social energy, talkativeness, creativity, leadership, stress handling, and more.

The goal of this project is to demonstrate **Machine Learning workflow including data preprocessing, EDA, feature engineering, model training, and evaluation.**

---
